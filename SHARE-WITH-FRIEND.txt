AmarMart - Setup for your friend
================================

This zip includes vendor/ (Composer packages).

1. Extract the zip.
2. Requirements: PHP 8.2+ with oci8 + pdo_oci, Oracle Instant Client, Oracle XE.
3. In the extracted folder:
   copy .env.example .env
   php artisan key:generate
4. Edit .env for your Oracle user/password/host.
5. Install DB objects (SQL*Plus):
   sqlplus USER/PASS@localhost:1521/XE @database/oracle/01_schema.sql
   sqlplus USER/PASS@localhost:1521/XE @database/oracle/02_plsql.sql
6. Run:
   php artisan storage:link
   php artisan db:seed
   php artisan serve

Open http://127.0.0.1:8000
